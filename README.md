# ♻️ RecycleVision - Garbage Image Classification Using Deep Learning

## Project Overview
RecycleVision is a deep learning based system that classifies garbage images into categories such as plastic, glass, metal, paper, cardboard, and trash.

The model uses **Transfer Learning with MobileNetV2** and is deployed using **Streamlit**.

---

## Objective
The main goal of this project is to automate waste classification using artificial intelligence.

• Build a deep learning model  
• Classify garbage images automatically  
• Improve recycling and waste management  
• Create a user-friendly dashboard

---

## Technologies Used

## How to Run the Project

Install dependencies

pip install -r requirements.txt

Run the model

python train.py

Run the dashboard

streamlit run app.py
